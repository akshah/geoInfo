Info

Our geolocation data is available using RESTful API. The data is obtained using BGP data from RouteViews and RIPE RIS, Infrastructure geolocation from CAIDA Ark Traceroutes, iPlane IP to PoP mappings and RIPE OpenIPMap, IXP participation from Packet Clearing House, PeeringDB and IXP websites from January 2016. Each query returns an array of JSON objects which can be read using any standard JSON parsing library. At this point we limit query rate to only 60 queries per second.

To view results in browser you can install JSONView extension.

Examples

General format: geoinfo.bgpmon.io/DATA_DATE/QUERY/VALUE
  • To get the list of all available data dates: geoinfo.bgpmon.io/data_dates
  • To get the latest data date: geoinfo.bgpmon.io/latest_data_date
Here are the list of queries we support:
  • curl geoinfo.bgpmon.io/201601/asn_country/12145
    • Shows country geolocation of the AS

  • curl geoinfo.bgpmon.io/201601/bgp_prefix_country/129.82.0.0/16
    • Shows country geolocation of the BGP prefix

  • curl geoinfo.bgpmon.io/201601/prefixes_announced_from_asn/12145
    • Shows all the prefixes announced from the AS

  • curl geoinfo.bgpmon.io/201601/prefixes_announced_from_country/KE
    • Shows all the BGP prefixes that originate ONLY from the country

  • curl geoinfo.bgpmon.io/DATA_DATE/bgp_prefixes_num_countries/[>,<,=]M
    • Shows BGP prefixes with [>,<,=] M countries in geolocation
    • geoinfo.bgpmon.io/201601/bgp_prefixes_num_countries/>2

  • curl geoinfo.bgpmon.io/DATA_DATE/slash24_prefixes_num_countries/[>,<,=]M
    • Shows /24 prefixes with [>,<,=] M countries in geolocation
    • geoinfo.bgpmon.io/201601/slash24_prefixes_num_countries/=3
